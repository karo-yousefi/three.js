[name]

This light globally illuminates all objects in the scene equally without considering their position or orientation.

This light cannot be used to cast shadows as it does not have a direction.

This type of light can be used to simulate indirect light or light bouncing. It affects every side of an object uniformly, illuminating it from all directions.


Code Example

const light = new THREE.AmbientLight( 0x404040 ); // soft white light scene.add( light );

In the example below, you can see the effect of AmbientLight on an object. The object appears flat, similar to a 2D image. This occurs because AmbientLight illuminates every surface equally without casting any shadows.



By adding another light to the scene, the effectiveness of this light in simulating indirect lighting becomes more apparent.
